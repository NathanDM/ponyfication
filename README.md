## Ponyfication

Simple electron application than allow you to manage your dev team.

Ponyfication's Component :

- [X] Team Item
- [X] Jenkins Api List Item
- [X] Firebase listener
- [ ] User List
- [ ] Mood Component
- [ ] Merge List Item


## Dev

### Install Dependencies

```sh
npm install
```

### First Build

```sh
gulp build
```

### Launch the App with Watcher (js, css, html, electron)

```sh
gulp
```
